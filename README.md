# Lemon Bucket

Personal Scoop bucket for [LemonUp](https://github.com/archcorsair/lemonup) and other tools.

## Usage

```powershell
scoop bucket add lemon-bucket https://github.com/archcorsair/lemon-bucket
scoop install lemonup
```

## Automated Updates

Manifests in this bucket are automatically updated via GitHub Actions (Excavator).
